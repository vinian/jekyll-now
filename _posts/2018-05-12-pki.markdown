## 名词解析
CA: Certificate Authority
CSR: Certificate Signing Request
DN: Distinguised Name
CN: Common Name
FQDN: Fully Qualified Domain Name

## 步骤

1. 生成 ca
2. 生成 key
3. 生成 csr
4. 对生成的 csr 采用 ca 签名
